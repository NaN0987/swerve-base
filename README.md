# swerve-base

This is a template used by 308 and 7178 for programming swerve drive for FRC. This template is designed to work with MK4/MK4i swerve modules with NEOs, CTRE CANcoders, and a NavX.

## Features
- Field-centric drive
- Odometry pose estimation
- Limelight 3 support
- Pathplanner support
